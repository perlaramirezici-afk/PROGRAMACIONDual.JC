¿Que es el SEO?

El SEO (por sus siglas en inglés, Search Engine Optimization) es el conjunto de estrategias y técnicas para optimizar un sitio web. Conjunto de estrategias y técnicas diseñadas para mejorar la visibilidad y el posicionamiento de un sitio web en los resultados orgánicos (no pagados) de buscadores como Google, Bing o Yahoo.
