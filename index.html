¿Qué es Responsive Design?

El diseño web responsivo adapta una página a cualquier dispositivo.

¿Qué es DOM ?

El DOM (Document Object Model) es una representación estructurada. Esto permite que los lenguajes de programación, como JavaScript, interactúen con la estructura, estilo y contenido del documento de manera dinámica.El DOM es esencial para crear páginas web dinámicas, ya que permite modificar elementos del documento en respuesta a eventos del usuario, como hacer clic en un botón, mover el ratón o escribir texto.

¿Qué es JavaScript?

lenguaje de programación ligero e interpretado que permite agregar interactividad y dinamismo a las páginas web una de las tecnologías fundamentales del desarrollo web, junto con HTML y CSS.Mientras que HTML estructura el contenido y CSS lo estiliza, JavaScript se encarga de implementar funcionalidades avanzadas como actualizaciones dinámicas, animaciones, control multimedia y mucho más.
